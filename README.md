# mobile_number_tracer
Here’s a clean, brief GitHub project description based on your code:

---

**Phone Number Info Lookup (Legal Use Only)**
A simple Python-based tool that extracts publicly available and legal metadata from phone numbers using the `phonenumbers` library. It provides details such as number validity, region, carrier, timezone, and number type in a user-friendly format. Designed for learning, analysis, and legitimate use cases—this tool does not perform real-time tracking or access private data.

----- how to run ---👇

---

## 🚀 1. Install Required Libraries

Open terminal / command prompt and run:

```bash
pip install phonenumbers requests
```

---

## 📁 2. Save Your File

Make sure your file is saved as:

```bash
phonenotracer.py
```

---

## ▶️ 3. Run the Project

Navigate to the folder where your file is located:

```bash
cd path/to/your/folder
```

Then run:

```bash
python phonenotracer.py
```

---

## 📱 4. Enter Phone Number

When prompted, enter number in **international format**:

Example:

```
+919876543210
```

---

## 📊 5. Output You’ll See(example)

You’ll get something like:

```
📱 Phone Information for +91 98765 43210
📍 Region: India
📶 Carrier: Airtel
⏰ Timezone: Asia/Kolkata
🔢 Type: Mobile
```

---

## ⚠️ Important Notes

* This tool **does NOT track live location**
* It only shows **publicly available telecom metadata**
* Works best with **valid international numbers**
